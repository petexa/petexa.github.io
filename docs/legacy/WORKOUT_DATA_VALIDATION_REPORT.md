# Workout Data Validation Report

> **Note:** This report is generated by the `clean_workouts.py` script. See [docs/workflows/WORKOUT_DATA_CLEANING.md](../workflows/WORKOUT_DATA_CLEANING.md) for information on running the script.

Validation reports are generated automatically during the workout data pipeline and contain statistics about:
- Rows processed
- Unit conversions applied
- Fields auto-filled via AI
- Duplicates found
- Contradictions detected
- Rows requiring manual review

New reports are written to `data/reports/` during pipeline runs.
